Laboratorio Virtual de Dinámica de Fluidos

Simulación interactiva de Reynolds, Poiseuille y Stokes

Introducción

Este recurso académico presenta algunos conceptos fundamentales de la dinámica de fluidos. El objetivo es analizar cómo variables físicas como el radio de un tubo o el tamaño de una partícula influyen en fenómenos como el caudal de un fluido o la velocidad de sedimentación.

Viscosidad

La viscosidad mide la resistencia de un fluido al movimiento.

Ecuación:

τ = μ (du/dy)

Variable Significado Unidad
τ Esfuerzo cortante Pa
μ Viscosidad dinámica Pa·s
du/dy Gradiente de velocidad s⁻¹

Número de Reynolds

Re = ρVD / μ

  • Re < 2000 → Flujo laminar
  • 2000 – 4000 → Transición
  • Re > 4000 → Turbulento

Uso de las barras deslizantes

Las barras deslizantes permiten modificar parámetros físicos del sistema de forma interactiva. Al mover la barra se cambia el valor de una variable, como el radio del tubo o el tamaño de una partícula, y el programa recalcula automáticamente el resultado utilizando las ecuaciones de la dinámica de fluidos.

Este tipo de herramientas permite explorar cómo influyen las variables físicas en los fenómenos del flujo sin necesidad de realizar un experimento real en laboratorio.
